# Reporting & Assessment Documentation

## Formal Reporting for Schools and IB Programmes

---

## Overview

Language learning progress needs to be communicated to multiple stakeholders in formats they expect and understand. This document specifies the reporting framework for LinguaFlow, covering school reports, IB requirements, and progress tracking.

---

## Report Types

| Report Type | Audience | Frequency | AI Assistance |
|-------------|----------|-----------|---------------|
| Semester Report | Parents, Students | 2x/year | Comment generation |
| IB MYP Report | Parents, IB Coordinator | 2x/year | Criterion analysis |
| IB DP Report | Parents, DP Coordinator | 2x/year | Component tracking |
| PT Interview Summary | Teachers | As needed | Data summary |
| CEFR Certificate | Students | On achievement | Automatic |
| Weekly Digest | Parents | Weekly | Fully automated |

## IB MYP Report Structure

### Criterion-Based Assessment (1-8)

| Criterion | Description | Weight |
|-----------|-------------|--------|
| **A** | Comprehending spoken and visual text | 25% |
| **B** | Comprehending written and visual text | 25% |
| **C** | Communicating in response to text | 25% |
| **D** | Using language in spoken and written form | 25% |

### MYP Phase Descriptors

| Phase | CEFR Equivalent | Year Level (typical) |
|-------|-----------------|---------------------|
| Phase 1 | A1 | Year 7 (beginner) |
| Phase 2 | A1-A2 | Year 7-8 |
| Phase 3 | A2 | Year 8-9 |
| Phase 4 | A2-B1 | Year 9-10 |
| Phase 5 | B1 | Year 10 |
| Phase 6 | B1-B2 | Year 10 (advanced) |

## AI Comment Generation

### How It Works

1. **Data Analysis**: System analyses student's LinguaFlow data
2. **Pattern Recognition**: Identifies strengths and areas for growth
3. **Draft Generation**: Creates personalised comments
4. **Teacher Review**: Teacher edits, approves, or regenerates
5. **Final Output**: Polished comments for reports

### Comment Bank Categories

**Strengths:**
- Listening comprehension
- Speaking confidence
- Reading fluency
- Writing organisation
- Engagement and effort

**Areas for Development:**
- Vocabulary range
- Grammar accuracy
- Pronunciation
- Fluency
- Written expression

## Summary

The reporting system ensures:

1. **Comprehensive Data**: All reports draw from LinguaFlow's rich learning data
2. **Curriculum Alignment**: Reports map to ACARA, IB MYP, and IB DP standards
3. **AI Assistance**: Teachers save time with AI-generated drafts
4. **Human Control**: All AI content is editable and approvable
5. **Multiple Formats**: PDF, digital portal, email digests
6. **Stakeholder Focus**: Right information for parents, students, coordinators
