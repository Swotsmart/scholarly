# LinguaFlow Gamification System

## Motivation Through Meaningful Progress

---

## Philosophy

Gamification in language learning must serve the learning, not replace it. Our system is designed around three principles:

1. **Intrinsic over Extrinsic**: Build genuine love of language, not just points addiction
2. **Progress Visibility**: Make invisible learning gains visible and celebratable
3. **Social Connection**: Learning together beats learning alone

## Configuration Options

### Teacher Controls

```typescript
interface TeacherGamificationConfig {
  // Enable/disable features
  enableLeaderboards: boolean;
  enableAchievements: boolean;
  enableXp: boolean;
  enableStreaks: boolean;
  enableChallenges: boolean;
  
  // Leaderboard settings
  leaderboardVisibility: 'full' | 'top_3' | 'relative' | 'off';
  leaderboardMetric: 'xp' | 'speaking_time' | 'streak' | 'custom';
  
  // Competition level
  competitionLevel: 'none' | 'low' | 'medium' | 'high';
  // none: No comparison, personal progress only
  // low: See own ranking, top 3 only
  // medium: Full class leaderboard
  // high: School-wide competition
  
  // Streak settings
  streakRequirementMinutes: number;  // Default: 5
  weekendStreakRequired: boolean;    // Default: false
  autoFreezeWeekends: boolean;       // Default: false
  
  // Challenge settings
  allowStudentChallenges: boolean;   // Students create challenges
  maxActiveChallenges: number;       // Default: 3
}
```

## Summary

The LinguaFlow Gamification System is designed to:

| Principle | Implementation |
|-----------|----------------|
| **Build habits, not addiction** | Flexible streak requirements, no punishment for breaks |
| **Reward effort, not just success** | XP for participation, not just high scores |
| **Make progress visible** | Clear XP progression, meaningful achievements |
| **Support all learners** | Opt-out options, anonymous modes, personal progress |
| **Connect to real learning** | Achievements tied to genuine language milestones |
| **Involve community** | Class challenges, co-op competitions, parent visibility |

The goal is students who **want** to practice languages, not students who **have** to maintain their streak.
