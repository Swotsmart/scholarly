## Summary

| Aspect | Traditional Approach | Heritage Pathway Approach |
|--------|---------------------|---------------------------|
| **Starting Point** | Assume zero knowledge | Assess existing skills |
| **Oral Skills** | Build from scratch | Validate and expand |
| **Literacy** | Secondary to oral | Primary focus |
| **Register** | Teach one register | Teach register shifting |
| **Dialect** | Correct to standard | Value and bridge |
| **Culture** | External content | Family-connected |
| **Assessment** | Uniform | Differentiated by gap |
| **Family** | Minimal involvement | Central to learning |

**The Goal**: Heritage speakers graduate with both their family's variety preserved AND formal language skills added â€” not one replaced by the other.
